Initiating SWGAide

Initialization is done in a few steps; any previous data is backed up to a file which is displayed at the log bar; see the README file on how to restore a DAT file.

If you use Windows Vista/7/10 you must allow Java to execute in "admin" mode, if you have not fixed that already you must exit now and see the README file, a section about Windows.

  1. Find and select the game folder that SWG lives on your computer that you want this instance of SWGAIde to work with, usually it is similar to C:\Program Files (x86)\StarWarsGalaxies, but you may have another setup depending on which emu/server you are using.
    Note:
  2. SWGAide builds a graphical view of your stations (accounts), galaxies, and characters that are found at your computer.

Once SWGAide is installed, always remember that  F1  always takes you to the help text for the current panel.

Many options and features are provided via popup menus that are opened by a right-hand mouse click at a GUI element.
