Welcome to SWGAide-Unity

Thanks for installing SWGAide-Unity that provides different kinds of features for Star Wars Galaxies.

Share your ideas and feedback on how to improve SWGAide.
Java programmers are welcomed to this open-source project.
Non-programmers are welcomed to gather and share any kind of data.
Welcome to https://swgaide.com/


First-time install: SWGAide must initiate itself, find SWG and the galaxies and characters you play; this is done by selecting your main game folder.

Otherwise: if you are a regular user but you see this screen and if you want to recover a backup file, exit SWGAide now and see the README file. Otherwise, continue.


Enjoy
