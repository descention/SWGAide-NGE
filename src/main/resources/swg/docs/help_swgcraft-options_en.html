Help: SWGCraft Options

These options apply to Traders and their support for swgaide.com. To use many of the crafting related features of SWGAide you must first register at www.swgcraft.co.uk.



Main galaxy: the default galaxy which is preselected at panels related to crafting, unless the current character is from another galaxy.

Auto-update: if this option is selected SWGAide automatically downloads resource data for the main galaxy.

Time zone: your time zone relative to the international GMT (UTC). Daylight saving is unsupported.

User name and password: your login details to swgaide.com.

Verify: validates the entered username/password before saving them locally. An invalid username and/or password makes many crafting related features of SWGAide unusable.

Save: the user name and password are the only data which you must save manually.

SWGCraft data: Details for communicating with the server. These details are locked and cannot be edited unless the check-box is ticked.

  1. The "base URL" to the server, the initial part of the link.
  2. The "resource path" is the specific path to the file for exported resource data. The "%BASE%" and "%GALAXY%.xml.gz" are filled in by SWGAide but the rest can be edited if ever necessary.
  3. The "status file" is the tiny file at the server that reads the most recent date the exported data files were updated. This file is always checked against to avoid excess downloads of the export files.
  4. SOAP is the path pointing to the SOAP server utilized by SWGAide to communicate resource data with swgaide.com.
Normally you should not have to touch these server settings and they are "locked" to prevent accidental data corruption.

If you completely erase the content of any of these fields it will reset to its default value when you restart SWGAide.
